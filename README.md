# Ulises_Proyecto_Individual
El proyecto es una pagina web que muestra los tipos de cambio de una moneda en específico
Se usa la página https://www.exchangerate-api.com/ para consultar los datos mediante su API, la version gratuita de su api se actualiza cada 24 hr

Para ejecutarlo solo se necesita conexión a internet y un naegador para abrir el archivo html

Instrucciones de uso:
En la parte superior de la página hay un boton para seleccionar el tipo de cambio, cada vez
que elegimos un tipo difernte la página hace una llamada a la API y actualiza la gráfica
